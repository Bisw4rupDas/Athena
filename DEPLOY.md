# Deploy COLOSSEUM (Athena)

## Render (recommended)

1. Push this repo to GitHub (`main` branch).
2. Open [Render Dashboard](https://dashboard.render.com/) → **New** → **Blueprint**.
3. Connect the `Athena` repository — Render reads `render.yaml` automatically.
4. Set environment variables when prompted:
   - `GROQ_API_KEY` — from [Groq Console](https://console.groq.com/)
   - `YOUTUBE_API_KEY` — from Google Cloud (YouTube Data API v3 enabled)
   - `SESSION_SECRET` — auto-generated by Render, or any long random string
5. Click **Apply** and wait for the deploy to finish.
6. Open your service URL (e.g. `https://colosseum.onrender.com`).

The persistent disk keeps `colosseum.db` and uploaded PDFs across restarts.

## Manual deploy on Render

1. **New** → **Web Service** → connect GitHub repo.
2. **Build command:** `npm install`
3. **Start command:** `npm start`
4. Add the same env vars as above.
5. Add a **Disk** (1 GB) mounted at `/opt/render/project/src`.

## Local production test

```bash
cp .env.example .env
# Fill in API keys in .env
set NODE_ENV=production
npm install
npm start
```

Visit `http://localhost:3000`.
